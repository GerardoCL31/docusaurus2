# Despliegue

El frontend esta preparado para hosting estatico, por ejemplo en One.com.

El backend puede desplegarse en Render u otros servicios compatibles con Node.js y MongoDB.

La separacion de despliegue mejora la flexibilidad y el mantenimiento del sistema.
