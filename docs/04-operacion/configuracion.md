# Configuración

El proyecto utiliza un archivo `.env` para centralizar parámetros de:

- Entorno.
- Persistencia.
- Credenciales.
- Seguridad.
- Telegram.

Este enfoque protege secretos y facilita un despliegue flexible en distintos entornos.
