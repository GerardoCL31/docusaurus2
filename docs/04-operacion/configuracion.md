# Configuracion

El proyecto utiliza un archivo `.env` para centralizar parametros de:

- Entorno.
- Persistencia.
- Credenciales.
- Seguridad.
- Telegram.

Este enfoque protege secretos y facilita un despliegue flexible en distintos entornos.
