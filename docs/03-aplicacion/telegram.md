# Integración con Telegram

CoronaHUB incluye notificaciones automáticas de reservas y opiniones mediante Telegram.

Las notificaciones incorporan botones de acción para confirmar o rechazar. Esto permite una administración inmediata desde un canal externo y demuestra una integración real con servicios de terceros.
