# Integracion con Telegram

CoronaHUB incluye notificaciones automaticas de reservas y opiniones mediante Telegram.

Las notificaciones incorporan botones de accion para confirmar o rechazar. Esto permite una administracion inmediata desde un canal externo y demuestra una integracion real con servicios de terceros.
