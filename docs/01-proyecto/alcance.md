# Alcance

## Área pública

El usuario puede acceder a:

- Pagina de inicio.
- Carta.
- Eventos.
- Galeria.
- Contacto.
- Informacion legal.
- Formulario de reservas.
- Formulario de opiniones.

## Área privada

El administrador puede:

- Iniciar sesión.
- Revisar, confirmar o cancelar reservas.
- Revisar, aprobar o rechazar opiniones.
- Editar carta, eventos, imágenes y textos desde el dashboard.

## Fuera de alcance

No forman parte de la versión actual:

- Registro de usuarios finales.
- Pasarela de pago.
- Email transaccional.
- Panel multiusuario.
- Analítica avanzada.
