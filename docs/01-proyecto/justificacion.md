# Justificacion

## Justificacion funcional

Un negocio como Bar Corona necesita una presencia digital que no dependa solo de redes sociales o llamadas telefonicas. CoronaHUB centraliza la informacion publica y aporta herramientas internas para operar con mas control.

Necesidades detectadas:

- Mostrar una presencia digital clara y actualizada.
- Publicar informacion sobre carta, eventos y datos del local.
- Gestionar reservas sin depender exclusivamente de llamadas o mensajes.
- Recoger opiniones y moderarlas antes de hacerlas visibles.
- Disponer de una zona interna para actualizar contenido y gestionar operaciones.

## Justificacion academica

El proyecto permite demostrar competencias completas de desarrollo full-stack porque integra:

- Interfaz de usuario.
- Navegacion y routing.
- Consumo de API.
- Backend REST.
- Persistencia de datos.
- Validacion de entradas.
- Seguridad y control de acceso.
- Pruebas automatizadas.
- Documentacion y despliegue.
