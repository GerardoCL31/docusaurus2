# Justificación

## Justificación funcional

Un negocio como Bar Corona necesita una presencia digital que no dependa solo de redes sociales o llamadas telefónicas. CoronaHUB centraliza la información pública y aporta herramientas internas para operar con más control.

Necesidades detectadas:

- Mostrar una presencia digital clara y actualizada.
- Públicar información sobre carta, eventos y datos del local.
- Gestionar reservas sin depender exclusivamente de llamadas o mensajes.
- Recoger opiniones y moderarlas antes de hacerlas visibles.
- Disponer de una zona interna para actualizar contenido y gestionar operaciones.

## Justificación académica

El proyecto permite demostrar competencias completas de desarrollo full-stack porque integra:

- Interfaz de usuario.
- Navegación y routing.
- Consumo de API.
- Backend REST.
- Persistencia de datos.
- Validación de entradas.
- Seguridad y control de acceso.
- Pruebas automatizadas.
- Documentación y despliegue.
