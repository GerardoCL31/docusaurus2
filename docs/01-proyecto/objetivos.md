# Objetivos

## Objetivo general

Desarrollar una aplicacion web full-stack para Bar Corona que mejore su presencia digital y facilite la gestion de reservas, opiniones, carta y eventos mediante una arquitectura mantenible, modular y funcional.

## Objetivos especificos

- Crear un frontend publico para clientes.
- Implementar un backend REST con separacion por dominios.
- Incorporar un formulario de reservas con validaciones de negocio.
- Implementar un sistema de opiniones con moderacion.
- Desarrollar un panel privado de administracion.
- Permitir la edicion de carta y eventos sin tocar codigo manualmente.
- Ofrecer persistencia configurable con MongoDB o archivo JSON.
- Aplicar autenticacion con JWT para la zona privada.
- Verificar el comportamiento critico mediante testing automatizado.
- Preparar el frontend para despliegue en hosting estatico.
