# Objetivos

## Objetivo general

Desarrollar una aplicación web full-stack para Bar Corona que mejore su presencia digital y facilite la gestión de reservas, opiniones, carta y eventos mediante una arquitectura mantenible, modular y funcional.

## Objetivos específicos

- Crear un frontend público para clientes.
- Implementar un backend REST con separación por dominios.
- Incorporar un formulario de reservas con validaciones de negocio.
- Implementar un sistema de opiniones con moderación.
- Desarrollar un panel privado de administración.
- Permitir la edición de carta y eventos sin tocar código manualmente.
- Ofrecer persistencia configurable con MongoDB o archivo JSON.
- Aplicar autenticación con JWT para la zona privada.
- Verificar el comportamiento crítico mediante testing automatizado.
- Preparar el frontend para despliegue en hosting estático.
